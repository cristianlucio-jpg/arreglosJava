# Proyecto: arreglosJava  
_Ejercicios prácticos de arreglos en Java_

## Descripción
Este proyecto contiene una serie de ejercicios en **Java** que aplican el manejo de **arreglos unidimensionales y bidimensionales** para resolver problemas básicos de lógica y programación.  
Forma parte de un portafolio académico y profesional orientado a la práctica de **estructuras de datos y algoritmos**.

---

## Objetivos
- Comprender y aplicar operaciones básicas sobre arreglos.  
- Utilizar técnicas de **búsqueda** y **ordenamiento** en colecciones de datos.  
- Desarrollar programas modulares y bien documentados con **código limpio**.  
- Practicar entrada y salida de datos con `Scanner` y `System.out`.  

---

## Contenido del proyecto

| Ejercicio              | Clase                     | Descripción |
|------------------------|---------------------------|-------------|
| **invertirVector**     | `invertirVector.java`     | Invierte el orden de los elementos de un arreglo. |
| **mayorElementoVector**| `mayorElementoVector.java`| Encuentra el valor máximo en un arreglo. |
| **matrizDiagonal**     | `matrizDiagonal.java`     | Trabaja con matrices y diagonales principales. |
| **sumarFilasColumnas** | `sumarFilasColumnas.java` | Suma elementos por filas y columnas en una matriz. |
| **urbanizacion**       | `urbanizacion.java`       | Aplica arreglos en un caso de urbanización. |
| **ventasTiendas**      | `ventasTiendas.java`      | Gestiona ventas en múltiples tiendas usando arreglos. |

---

## Contenido temático
- **Arreglos unidimensionales** → recorrido, búsqueda y ordenamiento.  
- **Arreglos bidimensionales** → operaciones con matrices.  
- **Técnicas de búsqueda** → lineal y condicional.  
- **Técnicas de ordenamiento** → básico (burbuja, selección).  
- **Aplicaciones prácticas** → urbanización, ventas en tiendas.
